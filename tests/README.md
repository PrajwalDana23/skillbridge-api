# SkillBridge API

## Overview
This is a FastAPI-based backend system for managing:
- User authentication (JWT)
- Batch & session management
- Attendance tracking
- Programme monitoring & summary

## Tech Stack
- FastAPI
- SQLAlchemy
- SQLite
- JWT Authentication
- Pytest

## Features
- User Signup & Login
- Role-based access (student, trainer, institution)
- Session creation
- Attendance marking
- Programme summary APIs
